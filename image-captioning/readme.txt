Deep Learning Photo Caption Generator from Scratch with Flicker8k dataset

steps:
1. generate image feature         (run -> python features.py)
2. clean description text         (run -> python descriptions.py)
3. define the model               (run -> python model.py)
4. pre-define tokenizer           (run -> python tokenizer.py)
5. evaluate the model             (run -> python evaluate.py)
6. generate caption (example.jpg) (run -> python generate.py)
